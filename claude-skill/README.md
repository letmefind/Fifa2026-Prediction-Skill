# Claude Skill Package

This folder contains a Claude-compatible skill for using the FIFA 2026 prediction engine.

Install or upload the nested skill directory:

```text
claude-skill/fifa2026-prediction/
```

The skill assumes Claude can access this repository or can clone it from:

```text
https://github.com/letmefind/Fifa2026-Prediction-Skill.git
```

See `fifa2026-prediction/SKILL.md` for the full skill instructions.
